# Тени Подземелья — modular isometric roguelite prototype

Мобильный HTML/WebGL-прототип на Three.js r128. Проект не использует import maps: Three.js и OrbitControls подключаются классическими глобальными `<script>`, локальная логика разделена на ES6-модули.

## Запуск

ES-модули нельзя надежно запускать через `file://`. Откройте терминал в корне проекта:

```bash
npm run serve
```

Затем откройте `http://localhost:8080`.

Для тестов генератора:

```bash
npm test
```

## Реализовано

- детерминированный Mulberry32 и хэширование строкового сида;
- scatter/separation комнат, немедленная переиндексация после culling;
- Bowyer–Watson Delaunay, Prim MST, гарантированная петля;
- вход, босс, сокровищницы, святилища, элитная арена и критический путь;
- коридоры шириной 1/2/3, стены, дверные клетки и flood-fill;
- безопасные спавны, сундуки, кристаллы, жаровни, колонны, факелы и мусор;
- точные InstancedMesh-буферы для сундуков/жаровен и пламени;
- ортографическая изометрия 45° / 35.264°, линейный туман и camera transitions;
- FOW: круг 2.5, конус 90° до 8.5, Bresenham LOS;
- пошаговое движение, Guard/Patrol/Wander/Idle, BFS-преследование;
- HP/PA/MA, Armello-подобные кости, взрывной Wyld и инициатива атакующего;
- карты: розыгрыш на карте и сжигание перед броском;
- добыча, экипировка, мобильный D-Pad и адаптивный HUD.

## Ограничения прототипа

Это вертикальный срез, а не законченная игра. В нем нет сохранений, аудио, анимаций скелета, полноценной кампании и серверной валидации. Количество врагов следует формуле из технического задания; на слабых устройствах разумно уменьшить `roomCount`.
